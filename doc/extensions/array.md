## Array extensions

### first()

Gives **access to the first element** in the array

````JavaScript
let names = ["Alice", "Bob", "Charlie"];
console.log(names.first()); // Output: "Alice"
````
outputs to 
````
Alice
````

### last()

Gives **access to the last element** in the array

````JavaScript
let names = ["Alice", "Bob", "Charlie"];
console.log(names.last()); // Output: "Charlie"
````
outputs to 
````
Charlie
````
